# Changelog

All notable changes to the NIHR Research Intelligence Dashboard project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-11-02

### Added
- Interactive Streamlit dashboard for NIHR funding data analysis
- Executive Summary with real-time KPIs and strategic metrics
- Advanced Data Quality Assessment with duplicate detection
- Parliamentary Analysis Tools for Southampton constituency
- Professional NIHR-branded UI/UX with responsive design
- Comprehensive missing value analysis and visualization
- Strategic funding intelligence and market opportunity analysis
- Automated Windows launcher script (`run_dashboard.bat`)
- Complete project documentation and setup instructions

### Features
- **Data Quality Assessment**: 81.2% completeness scoring system
- **Duplicate Detection**: Advanced pattern matching with 20.4% detection rate
- **National Benchmarking**: Southampton ranked #11/314 constituencies
- **Interactive Visualizations**: Plotly-powered charts and graphs
- **Parliamentary Insights**: Data-driven recommendations for MPs
- **Market Analysis**: ROI calculations and risk assessment framework

### Technical
- Streamlit frontend with custom CSS/HTML styling
- Pandas/NumPy for data processing and analytics
- Plotly for interactive data visualizations
- Professional documentation structure
- MIT License and comprehensive .gitignore
- Requirements.txt with all dependencies
